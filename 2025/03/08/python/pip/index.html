基础依赖包

NumPy: 用于科学计算的基础包。

Pandas: 数据分析和数据处理的库。

Requests: 简化 HTTP 请求的库。

Matplotlib: 绘图库，用于生成图表和可视化。

Scikit-learn: 机器学习库，提供各种算法和工具。

Flask: 轻量级的 Web 应用框架。

Django: 高级的 Web 应用框架，适合快速开发。

Beautiful Soup: 用于解析 HTML 和 XML 文档的库。

TensorFlow: 开源机器学习框架。

PyTorch: 另一个流行的深度学习框架。
